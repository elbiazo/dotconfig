# Dot Config

Contains .config for Windows and Linux

## Requirements

- Powershell. Yes it requires powershell on Linux as well :^)
- Git
- Python
- C Compiler (Need for Vim building tree-sitter)

### Windows

- Winget

## Setup

To setup entire environment, type `pwsh ./setup.ps1`
